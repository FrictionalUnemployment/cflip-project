server.js är backend
app.js är våran frontend. index.js är det som ska köra våran frontend

för att starta backend skriv 'node server.js' i /express_react_app katalogen
för att starta frontend skriv 'npm start' i /express_react_app/client katalogen

om crypto inte finns skriv 'npm install crypto --save' i /express_react_app katalogen
om express inte finns skriv 'npm install express --save' i /express_react_app katalogen
om frontend inte startar skriv 'npm install' i /experss_react_app/client katalogen
